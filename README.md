# Rental APP

Made With Laravel 10 + Vue Js with Inertia

PHP 8.2.13
Node 18.17.1

## How to Run
1. Setup Database
2. Run `composer install`
3. Run `npm install`
4. Run `php artisan key:generate`
5. Run `php artisan migrate`
6. For Test ~ Run `php artisan db:seed`
7. Run `npm run dev`
8. Run `php artisan serve`

Enjoy
